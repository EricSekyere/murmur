# AGENTS.md

Entry point for AI coding agents (and any automated tooling) working in this repository.

## Source of truth

All project conventions live in **[CLAUDE.md](./CLAUDE.md)**: tech stack, feature flags, build prerequisites, architecture principles, coding standards, testing, security, and commit style. Read it first and follow it exactly. This file only orients you toward it; when in doubt, CLAUDE.md wins.

See `prd.md` for product requirements and current implementation status.

## Quick orientation

- Cargo workspace, three crates:
  - `crates/murmur-core` — shared library (audio, STT, output, config, hotkeys)
  - `crates/murmur-cli` — CLI binary (clap)
  - `crates/murmur-app` — Tauri v2 desktop app
- Rust 1.93, edition 2024. Heavy native deps (whisper.cpp, ONNX Runtime) sit behind feature flags; default features stay light.

## Before you commit

- `cargo fmt --all`
- `cargo clippy --workspace -- -D warnings`
- `cargo test --workspace` (add `--all-features` when touching STT/VAD paths)
- Conventional Commit messages (`feat(core): ...`, `fix(audio): ...`).

## Hard rules

- Never include AI attribution (`Co-Authored-By`, "generated by", etc.) in commits, code comments, or any file.
- No `unwrap()`/`expect()`/`panic!` in production code; propagate errors.
- All processing local by default: no network calls or telemetry beyond model downloads.
